# Faster-RCNN

Implemented some  of  the  components  of  MaskRCNN,  an  algorithm  that addresses  the  task  of  instance  seg-mentation,  which  combines  object  detection  and  semantic segmentation into a per-pixel object detection framework.


In Part A of the Project:
Implemented a  Region  Proposal  Network  (RPN).

In Part B of the Project: 
Implemented and trained the object detection heads
